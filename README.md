# Akash Singh — Portfolio 🚀

A dark terminal-aesthetic portfolio site built with **React + Vite**, featuring smooth animations, modern UI elements, and sections for projects, research experience, skills, and more.

## 🔗 Live Demo

[akash-singh.vercel.app](https://portfolio-ak040.vercel.app/)

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| React 19 | UI framework |
| Vite 8 | Build tool & dev server |
| Framer Motion | Scroll animations & transitions |
| JetBrains Mono | Monospace / code font |
| Syne | Display / heading font |

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm

### Install & Run

```bash
# Clone the repo
git clone https://github.com/your-username/akash-portfolio.git
cd akash-portfolio

# Install dependencies
npm install

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

---

## 📦 Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder — deploy that to Vercel, Netlify, or GitHub Pages.

## ✨ Features

- Terminal / hacker aesthetic with green-on-dark color scheme
- Typewriter animation cycling through roles
- Scroll-triggered fade-in animations via Framer Motion
- Windows 11 style window title bars (minimize / maximize / close)
- Fully responsive — mobile, tablet, desktop
- Sections: Hero · About · Skills · Projects · Experience · Achievements · Contact

## 🌐 Deployment

Easiest way — connect your GitHub repo to [Vercel](https://vercel.com):

1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite — click Deploy
4. Done ✅
